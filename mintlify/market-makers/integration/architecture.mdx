## Component Overview

The Injective RFQ system enables retail traders to request competitive quotes from professional market makers,
with trades settled atomically on the Injective blockchain.
The system comprises four core components:

Component Technology Purpose
RFQ Contract Rust / CosmWasm On-chain quote verification, signature validation, trade settlement
RFQ Indexer Go / gRPC / MongoDB WebSocket streaming, request routing, quote aggregation, validation
Market Maker TS / Python / Go Connect via WebSocket, receive RFQ requests, sign & send quotes
Retail Client TS / Python / Go Submit RFQ requests, receive quotes, accept best quote on-chain
